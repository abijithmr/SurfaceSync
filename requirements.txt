# =============================================================================
#  QUICK INSTALL GUIDE
# =============================================================================
#
#  1. Make sure Python 3.10+ is installed:
#       python --version
#
#  2. (Recommended) Create a virtual environment first:
#       python -m venv venv
#       venv\Scripts\activate        # Windows
#       source venv/bin/activate     # Linux / macOS
#
#  3. Install all dependencies:
#       pip install -r requirements.txt
#
#  4. If TensorFlow install fails on older hardware try the lighter build:
#       pip install tensorflow-cpu==2.17.0
#
#  5. Run the backends (two separate terminals):
#       python transmitter_backend.py    # Terminal 1 - Laptop TX side
#       python receiver_backend.py       # Terminal 2 - Laptop RX side
#
#  6. Open the HTML frontends in your browser:
#       transmitter.html  ->  connects to http://localhost:5001
#       reciever.html     ->  connects to http://localhost:5002
#
